libVV
=====

libVV is a Volume Visualization library written by 盧瑞元 (Ruei-Yuan Lu).

It also includes a sample program called vvcli to demostrate all features.


Features
--------

* Hardware accelerated volume rendering and transfer functions
* Iso-surface extraction with hardware or software approach
* Support volume filtering


Dependencies
------------

* libgles2-mesa-dev
* ocl-icd-opencl-dev
